## Built With

## What I Learned

- How to structure complex layouts using **CSS Grid**
- Nesting grid containers within grid containers
- Using `grid-template-areas`, `fr` units and `grid-area` to control layout
- Working with SVG icons from Material Design
- Organizing HTML semantically with `aside`, `header` and `main`
